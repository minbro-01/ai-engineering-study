# results

이 폴더에는 실험 결과를 저장합니다.

예상 파일 예시:

```text
metrics.md              # 주요 지표 정리
confusion-matrix.png    # confusion matrix 이미지
training-log.md         # 학습 과정 기록
```

결과는 숫자만 남기지 않고, 어떤 조건에서 나온 결과인지 함께 기록합니다.

## Result Record Template

```text
Date:

Metric:

```
