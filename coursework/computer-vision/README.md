# Computer Vision

컴퓨터 비전 수업과 자율주행 인지 분야를 공부하며 정리하는 공간입니다.

수업자료나 외부 저장소를 그대로 옮기기보다, 핵심 개념을 내가 이해한 방식으로 다시 설명하고 작은 구현 아이디어로 연결하는 것을 목표로 합니다.

## Topics

| Topic | What to organize |
| --- | --- |
| Image Processing | filtering, edge detection, feature extraction 등 기본 처리 과정 |
| Object Detection | 객체 인식 모델의 구조, 학습 방식, 평가 지표 |
| Segmentation | semantic / instance segmentation 개념과 활용 사례 |
| Lane & Road Understanding | 차선, 도로 구조, topology 인식과 자율주행 인지 문제 |
| Model Evaluation | precision, recall, mAP, IoU 등 모델 평가 방식 |
