# Deep Learning

딥러닝 수업에서 다룬 RNN, CNN, Attention, Transformer 개념과 프로젝트 적용 아이디어를 정리하는 공간입니다.

## Topics

| Topic | What to understand |
| --- | --- |
| Neural Network Basics | layer, activation, loss, backpropagation |
| CNN | convolution, pooling, feature map, image recognition |
| RNN | sequence modeling, hidden state, vanishing gradient |
| Attention | sequence에서 중요한 정보를 선택하는 방식 |
| Transformer | self-attention, positional encoding, encoder/decoder 구조 |

## Study Questions

- RNN은 sequence 정보를 어떻게 hidden state에 저장하는가?
- Attention은 긴 sequence에서 어떤 문제를 해결하려고 등장했는가?
- Transformer 구조는 trajectory prediction 같은 시계열/sequence 문제에 어떻게 연결될 수 있는가?
- 모델 성능보다 먼저 확인해야 할 데이터 구조와 evaluation setting은 무엇인가?
