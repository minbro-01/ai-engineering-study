# Time Series Analysis

시계열 분석 수업과 관련된 개념, 모델, 실험을 정리하는 공간입니다.

통계 기반 방법부터 딥러닝 기반 시계열 모델까지 비교하면서 이해하는 것을 목표로 합니다.

## Topics

| Topic | What to understand |
| --- | --- |
| Time series basics | trend, seasonality, noise, stationarity |
| Statistical models | AR, MA, ARMA, ARIMA |
| Preprocessing | scaling, differencing, windowing |
| Evaluation | forecasting metric과 validation 방식 |
| RNN-based models | RNN, LSTM, GRU 기반 sequence modeling |
