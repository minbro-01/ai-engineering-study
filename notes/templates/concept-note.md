# Concept Note Template

```text
Date:
Course / Area:
Topic:

## Summary

## Key Concepts

## My Explanation

## Example

## Questions

```

## Writing Rule

- 수업자료를 그대로 옮기지 않습니다.
- 내가 이해한 방식으로 다시 설명합니다.
- 모르는 내용은 질문으로 남깁니다.
- 프로젝트와 연결되는 내용은 [`academic-projects`](https://github.com/minbro-01/academic-projects)에 연결합니다.
